FEIT - Call for Participation. Challenges in installing software development tools.

Struggling with installing new software development tools? We want to help!

Join our study to investigate the challenges developers face during tool installation and discover how Al can assist. Your participation will help improve the process for everyone!


Our study aims to analyse tools and comprehend the actions and decisions developers make when encountering problems during the process of installing software development tools. Additionally, we aim to investigate how Large Language Model Applications can assist software developers during this process. The discovery of these factors can help us understand how tools can be improved, making the installation process less troublesome for both newcomers and experienced developers in software development.


This participation is voluntary.

If you wish to participate, you will be asked to complete a questionnaire and participate in a think-aloud session (in person), where you will be asked to install a software development tool. The think-aloud session should not last more than one hour.


All students enrolled in a Computing or Information Systems major, and Masters of IT-related degrees at the University of Melbourne, are eligible for this study.


What's involved?

• Complete a brief questionnaire.

• Participate in a one-hour, in-person think-aloud session where you will be asked to install a software development tool.


Who can join?

• Students enrolled in Computing or Information Systems majors

• Masters of IT-related degrees (e.g., MCS, MIT, MSE) at the University of Melbourne

If you would like to participate, please fill in the form here: https://go.unimelb.edu.au/zc58
